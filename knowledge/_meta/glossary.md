# Glossary

- **Assistant layer:** Text reasoning, conversation, tools, and response orchestration.
- **Integration:** A provider-neutral capability such as weather or Home Assistant.
- **Provider adapter:** Code translating a neutral interface to a vendor SDK.
- **STT:** Speech-to-text.
- **TTS:** Text-to-speech.
- **VAD:** Voice activity detection.
- **Wake word:** Local phrase detection that starts voice capture.
