# Atlas

Atlas is a modular, self-hosted voice assistant intended to run on Raspberry Pi.
It coordinates replaceable voice, assistant, and integration layers while
keeping provider SDKs at the edges.

Atlas currently supports configuration validation and a text-first model seam.
Voice processing and external integrations are later milestones.

See [System architecture](../guides/system-architecture.md).
